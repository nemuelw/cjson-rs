# cjson-rs

Rust bindings for the cJSON library

## Setup

Make sure you have the [cJSON](https://github.com/DaveGamble/cJSON) library properly installed your system.

- Update your `Cargo.toml` file:

    ```toml
    cjson-rs = "0.1.0"
    ```
- Import the crate to start using it:

    ```rust
    use cjson_rs::*;
    ```
